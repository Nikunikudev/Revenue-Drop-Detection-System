# Revenue Drop Detection System

## Overview

An automated Revenue Drop Detection System built using AWS Lambda, Amazon S3, Python, and Power BI.

The solution automatically analyzes sales data, detects significant revenue declines, generates alerts, and provides executive-level insights through an interactive dashboard.

## Technology Stack

- Python
- AWS Lambda
- Amazon S3
- Power BI
- DAX
- SVG Visuals

## Architecture

Sales Data CSV
↓
Amazon S3
↓
AWS Lambda
↓
alerts.csv
↓
Power BI Dashboard

## Features

- Automated Revenue Drop Detection
- Revenue At Risk Analysis
- Critical Alert Generation
- Smart Alert Insights
- Executive Summary
- Revenue Exposure Tracking
